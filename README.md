용어 정리
1. 테스크  :  
 컴퓨팅에서 태스크(task)는 작업 단위의 실행 단위이다.  
  프로세스, 경량 프로세스, 스레드(실행을 위해), 스텝, 요청, 쿼리(작업을 위해)가 있다  
  작업 단위 그 자체나 해당 작업을 수행하는 스레드를 "태스크"라고 부를 수 있으며 이것들은 각각 요청/응답/스레드, 들어오는 태스크/완료된 태스크/스레드(그림에 따라) 또는 요청/응답/태스크로 부를 수 있다.
